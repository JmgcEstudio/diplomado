# PROYECTO DIPLOMADO 
### 1.- Clonar el Proyecto
```bash
https://github.com/JmgcEstudio/diplomado.git
```
### 2.- Instalar dependencias
```bash
npm instal
```
### 3.- Copiar las variables de entorno y configurar
```bash
cp .env.sample .env
```
### 4.- Levantar el Proyecto
#### 4.1.- En desarrollo
```bash
npm rum dev
```
#### 4.1.- En produccion
```bash
npm rum star
```
